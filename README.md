# Índices ESTAR (v2 - agrupado)
Ahora el shortcode se asocia a un **Índice (grupo)**, y dentro de ese índice se navega por sus **Números**.

## Shortcodes
- `[indices_estar group="1"]`
- `[indices_estar slug="mi-indice"]`

## Admin
Menú: **Índices**
- Pantalla principal: listado de índices (grupos) con shortcode copiable.
- “Gestionar números”: CRUD de números y sus contenidos.

## Migración
Al activar, se crea la tabla de grupos y se asignan los datos existentes al grupo por defecto.
